#  Fichiers d'un exercice

En plus d'éventuels fichiers de ressources (images, sons, etc.), un exercice est composé de plusieurs fichiers importants et obligatoires pour certains :

- **main.ple** (Point d'Entrée de l'Exercice)
  - Fichier principal contenant le code de l'exercice.
- **main.plf** (Optionnel)
  - Fichier pour créer un formulaire graphique via editor.js.
- **config.json** (Optionnel)
  - Fichier de configuration des variables de l'exercice.
- **readme.md**
  - Document explicatif de l'exercice, à usage des enseignants.


Lorsque vous créer un exercice, ces fichiers sont automatiquement générés avec un exemple fontionnel de base. Vous pouvez ensuite les modifier à votre guise
et supprimer ceux qui ne vous sont pas utiles.
